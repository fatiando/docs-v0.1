The fatiando package contains all the subpackages and modules required for most tasks.

Modules for each geophysical method are group in subpackages:

  • gravmag: Gravity and magnetics (i.e., potential fields)
  • seismic: Seismics and seismology
  • geothermal: Geothermal heat transfer modeling

Modules for gridding, meshing, visualization, user interface, input/output etc:

  • mesher: Mesh generation and definition of geometric elements
  • gridder: Grid generation and operations (e.g., interpolation)
  • vis: Plotting utilities for 2D (using matplotlib) and 3D (using mayavi)
  • io: Input/Output of models, data sets, etc (fetch from web repositories)
  • gui: Graphical user interfaces (still very primitive)
  • utils: Miscelaneous utilities, like mathematical functions, unit conversion, etc
  • logger: A simpler interface to the Python logging module for log files
  • constants: Physical constants and unit conversions

Also included is the fatiando.inversion package with utilities for implementing inverse problems. There you’ll find common regularizing functions, linear inverse problem solvers, and non-linear gradient solvers. This package is generaly only used from inside Fatiando itself, not when using Fatiando in scripts. For usage examples, see the source of modules fatiando.seismic.epic2d and fatiando.gravmag.basin2d.

See the documentation for each module to find out more about what they do and how to use them.

  • Gravity and magnetics (fatiando.gravmag)
    • 3D inversion by planting anomalous densities (fatiando.gravmag.harvester)
    • Forward modeling with 3D right rectangular prisms (fatiando.gravmag.prism)
    • Forward modeling with 3D polygonal prisms (fatiando.gravmag.polyprism)
    • Forward modeling with spheres (fatiando.gravmag.sphere)
    • Forward modeling with tesseroids (fatiando.gravmag.tesseroid)
    • Forward modeling with 2D polygons (fatiando.gravmag.talwani)
    • Forward modeling of half a spherical shell (fatiando.gravmag.half_sph_shell)
    • Inversion for the relief of 2D basins (fatiando.gravmag.basin2d)
    • Processing and transformations using the FFT (fatiando.gravmag.fourier)
    • Imaging methods for potential fields (fatiando.gravmag.imaging)
    • Utilities for operating on the gradient tensor (fatiando.gravmag.tensor)
    • 3D Euler deconvolution methods (fatiando.gravmag.euler)
    • Space domain processing (fatiando.gravmag.transform)
  • Seismics and seismology (fatiando.seismic)
    • 2D seismic ray travel-time modeling (fatiando.seismic.ttime2d)
    • 2D epicenter determination (fatiando.seismic.epic2d)
    • Seismic profiling (fatiando.seismic.profile)
    • Straigh-ray 2D tomography with SrTomo (fatiando.seismic.srtomo)
    • Finite difference solution of the 2D wave equation (fatiando.seismic.wavefd)
  • Geothermal heat (fatiando.geothermal)
    • Climate change signal in well temperature logs (fatiando.geothermal.climsig)
  • Meshing (fatiando.mesher)
  • Gridding (fatiando.gridder)
  • Visualization (fatiando.vis)
    • Plot 2D objects, maps and grids with matplotlib (fatiando.vis.mpl)
    • Plot 3D objects in Mayavi2 (fatiando.vis.myv)
  • Input/Output (fatiando.io)
  • Graphical user interfaces (fatiando.gui)
    • Simple GUIs using matplotlib (fatiando.gui.simple)
  • Miscellaneous Utilities (fatiando.utils)
  • Logging (fatiando.logger)
  • Physical constants and unit conversions (fatiando.constants)
  • Inverse problem solving tools (fatiando.inversion)
    • Solvers for linear problems (fatiando.inversion.linear)
    • Gradient solvers (fatiando.inversion.gradient)
    • Heuristic solvers (fatiando.inversion.heuristic)
    • Regularizing functions (fatiando.inversion.regularizer)
    • Base data module class (fatiando.inversion.datamodule)
